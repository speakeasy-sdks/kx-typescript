# Ai
(*ai*)

### Available Operations

* [insertRaw](#insertraw) - Insert data into a table.
* [insertJson](#insertjson) - Insert data into a table.
* [trainRaw](#trainraw) - Train index on the table (IVF and IVFPQ only).
* [trainJson](#trainjson) - Train index on the table (IVF and IVFPQ only).
* [vectorSearch](#vectorsearch) - Vector similarity search
* [hybridSearch](#hybridsearch) - hybrid similarity search

## insertRaw

Request body is a tuple of (target table name, data table) serialized into a PyKX octet stream with pykx._wrappers._to_bytes(6, pykx.toq([table, data]), 1)[1])

### Example Usage

```typescript
import { Kx } from "kx";

const kx = new Kx({
  apiKeyAuth: "<YOUR_API_KEY_HERE>",
});

async function run() {
  const result = await kx.ai.insertRaw(new TextEncoder().encode("0x02eF1eFB84"));

  // Handle the result
  console.log(result)
}

run();
```
