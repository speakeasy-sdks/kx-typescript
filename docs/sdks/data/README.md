## query

Perform query to get data from a table

### Example Usage

```typescript
import { Kx } from "kx";

const kx = new Kx({
  apiKeyAuth: "<YOUR_API_KEY_HERE>",
});

async function run() {
  const result = await kx.ai.data.query({
    table: "<value>",
  });

  // Handle the result
  console.log(result)
}

run();
```
